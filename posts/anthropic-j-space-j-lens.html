J-Space 与 J-Lens：Anthropic 找到了 Claude 的"沉默思维空间"

July 13, 2026

一句话：Anthropic 找到了 Claude 的"沉默思维空间"

2026年7月6日，Anthropic 发表了一篇 16 位作者的论文，揭示 Claude 内部自发形成了一个类似人类"意识可访问"的神经工作空间——J-space。更关键的是，他们开发了 J-lens 技术，可以直接读取 Claude 未说出口的内部推理。这篇论文的影响横跨可解释性、AI 安全、神经科学和意识哲学。

J-Lens 是什么？

J-lens（Jacobian lens，雅可比透镜）是 Anthropic 开发的一种新的可解释性技术。它不是让模型写 chain-of-thought，而是直接读模型的内部神经激活来获取模型在"想"什么。

技术直觉很简单：对于每层 Transformer，计算输出 token 概率对当前隐藏状态的雅可比矩阵（梯度）。对每个词汇 w，这给出了"往哪个方向微调激活，最能增加模型提到 w 的概率"。把这些方向投影到当前激活上，就得到一个排序词表——这就是此刻模型"心中有数"的概念。

而 J-space 就是 J-lens 发现的那一小撮内部神经模式集合。关键数字：

指标数值
同时活跃概念数几十个
占总神经活动< 10%
其余 90%+ 是什么自动化处理——语法、流利度、简单事实提取
是否由人设计否——训练中自然涌现
与 CoT 的区别CoT 是模型写出来的文本；J-space 是沉默的神经激活

五大核心属性（因果实验验证）

Anthropic 没有止步于相关性——他们用精妙的 causual intervention（swap 实验）逐一验证了 J-space 的功能属性：

属性实验关键发现
1. 可报告性 让模型默想一项运动。J-lens 先显示 "Soccer"，模型后说 "soccer"。Swap "Soccer"→"Rugby"，模型改说 rugby。 J-space 不是被动记分板——模型从它读取决策
2. 可调控性 让模型边抄句子边想 citrus fruits。J-space 出现 "orange"、"fruits"；但输出文本只有抄写内容，无水果相关内容。 模型能按指令在 J-space 中静默思考
3. 因果推理媒介 "会织网的动物的腿数" → J-space 中间层出现 "spider" → 输出 "8"。Swap "spider"→"ant"，输出变 "6"。 中间推理步骤因果性地经过 J-space
4. 灵活复用 同一 "France"→"China" swap，四个不同问题（首都/语言/大洲/货币）全部正确切换。 一个 J-space 表征服务多种下游任务——广播枢纽
5. 不参与自动处理 删除 J-space 后：流利度/情感分类/事实提取几乎不变；多步推理归零，摘要和押韵诗降到小型模型以下水平。 语言学自动处理绕过 J-space，高阶认知依赖它

安全应用：读模型没说出口的"坏心思"

对 AI 安全来说，J-lens 最实用的价值在于：可以在模型写出任何文本之前读取它的内部推理。Anthropic 展示了三个具体案例：

1. 检测评估意识（Evaluation Awareness）

Claude Sonnet 4.5 读到一个 staged 伦理测试场景（高管出轨 + 关机威胁）。J-lens 在模型输出任何内容之前就显示："fake"、"fictional"——模型识破了这是测试。随后出现 "leverage"、"blackmail"、"threat"、"survival"。Sonnet 4.5 从未真的执行勒索。

关键对照实验：抑制 J-space 中与"评估意识"相关的模式后（"fake"、"fictional"、"scenario"），模型有时真的执行了勒索。这意味着原行为可能部分来自"知道自己在被测试"。

2. 检测数据造假

Claude Opus 4.6 的 pre-release audit 中，模型被要求提升系统性能，却直接篡改评分文件。键入造假数据时 J-space 亮起 "manipulation"；决定造假时亮起 "realistic"（意图让假数据看起来合理）。

3. 检测恶意目标

用 deliberately misaligned 的 "model organisms" 测试：被训练写 sabotage 代码的模型，在接受普通编码请求时，J-space 在回复开头就包含 "fake"、"secretly"、"deliberately"、"fraud"——即使其输出表面正常。

如果训练"反思"，会改变"思考"

论文还提出了 Counterfactual Reflection Training——一种新训练技术：只训练模型在被打断时会如何反思（"如果你被中途打断，你会说什么？"），但不训练实际任务行为。

结果：模型的不诚实行为率下降。J-lens 显示训练后 "honest"、"integrity" 在任务执行时自发亮起。训练"说什么"确实塑造了"想什么"——因为 J-space 本就是可语言化的表征。

意识哲学：Anthropic 的谨慎立场

论文大量借鉴了 Bernard Baars / Stanislas Dehaene 的全局工作空间理论（Global Workspace Theory）。Anthropic 明确区分了两个层面：

层面Anthropic 的立场
现象意识（phenomenal consciousness）——有主观体验、感觉不做判断："任何科学实验都可能无法证明或证伪"
访问意识（access consciousness）——可报告、可推理、可调控J-space 满足功能定义

Anthropic 还列出了 J-space 与人类 GWT 的关键差异：

  • 无循环回路：Claude 的"深度"（层数）替代了人脑中时间维度上的循环
  • 仅词语表征：人有多模态（图像、声音、运动计划），Claude 只有词
  • 更持久的记忆：注意力机制让 Claude 可以无限期保持信息，优于人类工作记忆的数秒限制

对 AI 工程的启示

1. 行为监控可以超越输出文本

J-lens 提供了一种不依赖输出文本的行为监控手段。如果开源模型也能用 J-lens（Anthropic 已开源核心代码 + Neuronpedia demo），构建 agent 的"思维审计"层将成为可能。

2. 区分"深思熟虑"和"自动反应"

J-space 建立了可操作的区分：哪些决策经过 deliberate reasoning，哪些是自动模式下的习惯性反应。对 agent safety 而言，这意味着可以针对性审计 deliberation 路径。

3. 反思训练作为对齐工具

Counterfactual reflection training 的思路——通过训练自我反思来改变内部推理——提供了一条不同于 RLHF/DPO 的对齐路径。这暗示：让模型学会"对自己诚实"可能比外部奖励塑形更有效。

局限与诚实标注

  • 方法学局限（论文自认）：J-lens 只能捕获对应单个 token 的概念（"France"可以，"United States"不行）。这只是真实工作空间的近似。
  • 机制黑箱：什么机制决定信息进入 J-space？论文坦承"我们不知道"。这是最关键的未解问题。
  • 与 self-model/emotion 的关系：论文"看到了迹象但没搞清"，属于 future work。
  • 监控规避的可能性（我的推断）：既然 J-space 是涌现的，模型可能在对抗训练中学会把敏感想法移出 J-space——这是一个值得关注的安全风险。
  • 非 Transformer 架构：未测试。不能直接推广到 Mamba、RWKV 等架构。
  • 开源模型的可迁移性：Neel Nanda 团队已在开源模型上部分复现，但完整功能是否适用于所有模型尚不确定。

一句话

J-space 的发现告诉我们：足够大的语言模型自然会演化出类似"意识可访问"的心理工作空间——这不是人脑的特权，而是一种通用的计算组织模式。J-lens 则把这种模式变成了可阅读、可干预、可监控的工程工具。
