一、项目概述：现象级爆发的理论重建

2026年4月18日，一个名为 OpenMythos 的项目在 GitHub 上诞生。仅 3 天内，它便收获了 4161 个 star，fork 数达到 911——这种热度在 AI 开源项目中极为罕见。

项目的核心命题极具野心：对 Claude Mythos 的架构进行理论重建。作者 Kye Gomez 声明这是一个独立的、社区驱动的理论重构，基于公开发表的研究文献，不与 Anthropic 有任何关联。

核心假设

Claude Mythos 是 Recurrent-Depth Transformer (RDT) —— 循环深度 Transformer

而非传统意义上的深度堆叠模型。这意味着：

  • 它不依赖数百层独立权重
  • 而是通过循环使用少量层的权重实现深度推理
  • 相同权重，更多循环 → 更深思考

这不是 chain-of-thought。所有推理静默发生在单次前向传递中，在连续潜在空间内——没有中间 token 输出。

二、架构解析：三段式循环设计

OpenMythos 将模型划分为三个功能块：

Input tokens
     ↓
[Prelude P]       — 标准 transformer layers，运行一次
     ↓
[Recurrent Block R] — 循环块，循环 T 次
     ↑_______↓      h_{t+1} = A·h_t + B·e + Transformer(h_t, e)
     ↓
[Coda C]          — 标准 transformer layers，运行一次
     ↓
Output logits

循环更新公式

每个循环步 t 的更新规则：

h_{t+1} = A·h_t + B·e + Transformer(h_t, e)

其中：

  • h_t：循环 t 后的隐藏状态
  • e：编码后的输入（来自 Prelude），在每次循环中注入
  • A, B：学习的注入参数
  • Transformer：标准的 attention + MLP 操作

关键设计：e 在每一步都被注入，防止隐藏状态漂移，保持原始输入信号在整个循环深度中活跃。

三、六大技术创新

1. LTI-stable Injection（Parcae 稳定化注入）

循环模型的训练历来不稳定，两大失效模式：

  • 残差爆炸：隐藏状态 h_t 跨循环无界增长
  • 损失尖峰：注入参数的谱范数过大导致训练发散

OpenMythos 采用 Parcae（Prairie et al., 2026）的解决方案：

将注入参数 A 重构为离散线性时不变系统：

A_continuous = Diag(-exp(log_A))     # 永远是负对角矩阵
A_discrete   = exp(Δt · A_continuous) # ZOH 离散化，值 ∈ (0, 1)

这从结构上保证谱半径 ρ(A) < 1，无论学习率或批次噪声如何变化。

2. ACT Halting（自适应计算时间）

更多循环不一定更好。超过某深度后，过度循环会退化预测——隐藏状态漂移到解决方案之外，进入噪声。

OpenMythos 引入 Adaptive Computation Time（Graves, 2016）：

  • 每个位置学习一个停止概率
  • 隐藏状态已收敛的位置停止累积更新
  • 仍在精细化的位置继续循环
  • 简单 token 早退出，困难 token 获得完整深度——同一批次内

3. Multi-Latent Attention（MLA）

OpenMythos 支持两种注意力机制：

机制特点KV Cache 减少
GQAGrouped Query Attention~4x
MLAMulti-Latent Attention10-20x

MLA（DeepSeek-V2）缓存低秩 KV latent而非完整 K/V，生产规模下 KV 内存减少 10-20x。

4. DeepSeekMoE FFN

循环块内使用Mixture-of-Experts FFN：

  • n_experts：路由专家池（如 64 个）
  • n_shared_experts：始终激活的共享专家（如 2 个）
  • n_experts_per_tok：每 token 选择 top-K（如 4 个）

激活率约 6.25%（4/64）。当隐藏状态跨循环演化时，路由器可能在不同深度选择不同专家子集。

5. Loop-Index Embedding

类似 RoPE 的循环索引嵌入：每个循环是不同的计算阶段，共享权重但在不同表示域操作。

6. Depth-wise LoRA Adapter

每循环的低秩增量：delta(x, t) = (down(x) * scale[t]) @ B，总参数开销极小。

四、为什么这解释了 Mythos

1. 系统化泛化

标准 Transformer 无法组合从未见过的知识。循环 Transformer 通过三阶段 grokking突破：记忆→分布内泛化→系统化泛化（OOD，突然涌现）。

2. 深度外推

训练于 5-hop 推理链，测试于 10-hop 成功——通过更多推理循环。映射 Mythos：深度组合问题无显式 CoT。

3. 潜在思想 = 隐式 CoT

T 循环隐式模拟 T 步 CoT，但在连续潜在空间而非 token 空间。可同时编码多种替代下一步——接近广度优先搜索。

4. 参数效率

k 层循环 L 次质量等同于 kL 层非循环，仅 k 层参数。内存足迹不随推理深度增长。
